# Virtual Painting System

A real-time virtual painting application, also known as **Air Canvas**, that uses hand gesture detection to draw on the screen without physical contact.

## Features

* Draw using index-finger movement
* Select different colors
* Clear the drawing canvas
* Real-time hand tracking through a webcam

## Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy

## How to Run

1. Clone the repository.
2. Create and activate a virtual environment.
3. Install the required packages:

```bash
pip install mediapipe==0.10.21 opencv-python numpy
```

4. Run the program:

```bash
python main.py
```

5. Press `q` to close the application.

## Requirements

* Python 3.11
* Webcam
